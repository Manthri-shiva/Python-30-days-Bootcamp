# 🎯 Bootcamp Objective

This repository documents my **30-Day Python Bootcamp**, where I will learn Python from scratch while building strong programming fundamentals, developing problem-solving skills, and preparing for technical interviews.

The goal is to become confident in:

* Python Fundamentals
* Object-Oriented Programming
* Data Structures & Algorithms
* Problem Solving
* Python Best Practices (PEP 8)
* APIs & Databases
* Flask & FastAPI
* Git & GitHub
* Technical Interviews
